## URURL 팀은,,
총 7명으로 이루어진 팀으로, 자신의 링크를 효율적으로 관리하고자 하는 니즈를 충족시키기 위해 이루어진 팀입니다.

웹, 웹 익스텐션, 앱까지 확장할 발전을 노리고 있는 팀입니다.
